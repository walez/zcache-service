# Zen Cache

A cache web service that implements caching using LRU

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start:dev

# watch mode
$ yarn run start:watch
```

## Test

```bash
# unit tests
$ yarn run test
```
